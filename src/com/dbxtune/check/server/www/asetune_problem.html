Known issue in release 2012-04-20 (3.0.0)

Problem: In the Connection Dialog, the "OK" button might not be visible

Solution: Simply resize the window.
The "saved/restored" window size was simply to small
This will only happen to users that has upgraded from any earlier releases.

This issue will be fixed in next release.
Solution: On dialog open: The initial minimum window size will be enforced.
