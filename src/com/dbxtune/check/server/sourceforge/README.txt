==================================================
Pre requirement:
--------------------------------------------------
Java 6, or later installed on the machine

==================================================
Installation:
--------------------------------------------------
On any platform:
- simply unzip the zip file into a directory

==================================================
Start the tool:
--------------------------------------------------
Windows:
- asetune.bat

Unix/Linux:
- asetune.sh

==================================================
Connect to ASE Server
--------------------------------------------------
- Alt+o or Menu->File->Connect or Icon at the toolbar

Then start to view Performance Counters in the tool
